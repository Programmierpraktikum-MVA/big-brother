OpenCV

```bash
  pip install opencv-python
```
Tesseract 
```bash
The installation proceeds differently depending on the operating system.
```

```bash
pip install nltk
```

```bash
pip install rake-nltk
```

Tkinter
```bash
pip install tk
```

VLC
```bash
pip install python-vlc
```
